About Vardonit

Vardonit is a simple social posting platform built from scratch using HTML, CSS, JavaScript, and Node.js.

The idea was inspired by Zuckonit — the fictional mini-blog from The Social Network, where a young Mark Zuckerberg posted raw, unfiltered thoughts online before building Facebook. Vardonit brings that same raw energy back.

What you can do:
Post your thoughts · Like posts · Delete posts · See timestamps and names on every post.

Built with:
HTML · CSS · JavaScript · Node.js · Express · JSON storage
